It focused on selecting suitable AI models and designing the foundational architecture of the EduGenie Learning Assistant. The objective of this activity was to ensure efficient educational assistance, modular development, and smooth integration between frontend, backend, and AI-powered functionalities. For advanced educational functionalities such as Question Answering (Q&amp;A), Summarization, Quiz Generation, and Personalized Learning Recommendations, Google Gemini 1.5 Pro was selected through API integration. Gemini 1.5 Pro was chosen because of its advanced reasoning capabilities, contextual understanding, structured response generation, and cloud inference support. The model enables EduGenie to generate intelligent and human-like educational content while maintaining high accuracy and efficiency across multiple learning tasks. For concept explanation functionality, LaMini-Flan-T5-783M was selected as the local lightweight AI model. This model is instruction-tuned and specifically suitable for generating simplified explanations for complex educational topics. Since the model is CPU-compatible and lightweight, it improves local execution efficiency and reduces dependency on cloud resources for simpler tasks. The model helps learners understand difficult concepts in a concise, beginner-friendly, and highly readable manner. Along with model selection, a modular folder architecture was designed to organize the project efficiently. The architecture separated backend modules, frontend resources, AI functionalities, and dependency management into independent files for better scalability, maintainability, and debugging. The folder structure included dedicated Python modules such as: qna.py for Question Answering explanation_module.py for concept explanation quiz_module.py for quiz generation summary_module.py for summarization learning_path.py for personalized learning recommendations The main.py file acted as the central FastAPI application responsible for routing and backend integration. Frontend resources were managed through the templates and static folders containing index.html and style.css, respectively. Additionally, the requirements.txt file stored all the required Python dependencies needed to run the EduGenie application successfully. Explanation Module: EduGenie utilizes the LaMini-Flan-T5 model, a lightweight yet powerful generative AI, to deliver educational content in a simplified and highly readable manner. This model is specifically fine-tuned to provide concise, context-aware responses that break down complex topics into easily understandable language. By focusing on clarity and brevity, EduGenie ensures that learners with minimal background knowledge or technical experience can grasp essential concepts without feeling overwhelmed. This makes it particularly valuable for beginners, school students, or self-learners seeking straightforward explanations. The integration of LaMini-Flan-T5 empowers EduGenie to act as a reliable and accessible study companion for foundational learning across subjects. QnA Module: EduGenie is powered by Gemini 1.5 Pro, enabling it to handle a wide range of general knowledge and academic question-answering tasks with precision. Its advanced understanding and contextual capabilities make it an effective tool for learners seeking accurate, AI-driven assistance across diverse educational topics and subjects. Quiz Module: EduGenie generates three multiple-choice questions (MCQs) from a given passage, each containing four carefully crafted options. It utilizes the Gemini model to comprehend the context and meaning of the passage, ensuring that the questions are relevant and the distractors are plausible. The final output is structured in JSON format for easy integration. It sends a structured prompt to the model, expecting a valid JSON response with each question containing four options and a correct answer. To ensure proper parsing, it cleans any Markdown code blocks using the clean_json_blockfunction . The cleaned response is then parsed into a Python list. If an error occurs during generation or parsing, it returns a detailed error message for debugging. Summary Module: This feature leverages Gemini’s generative capabilities to summarize long paragraphs into concise, easy-to-understand versions, making it ideal for quick revision. It ensures the core information is retained while eliminating redundancy, helping learners absorb key points efficiently without losing clarity or context. Learning Path Module: The get_learning_recommendations function uses Google Gemini to generate a personalized, structured learning path for any given topic. Crafting a detailed prompt, it instructs the AI to suggest beginner to advanced concepts, organized by difficulty, and supported with useful resources such as videos, articles, or books. It ensures adaptability to the learner’s level. The function safely handles API responses and errors, providing either the generated text or a helpful error message if the response is invalid or incomplete. It focused on developing the backend infrastructure of the EduGenie Learning Assistant using the FastAPI framework. In this activity, RESTful API endpoints were created for each educational functionality so that the frontend interface could communicate efficiently with the AI-powered backend modules. FastAPI was selected because of its high performance, asynchronous request handling, simplicity, and easy API integration capabilities. The backend architecture was designed in a modular manner, where each endpoint was connected to its respective AI functionality. Separate API routes were developed for Question Answering, Concept Explanation, Quiz Generation, Summarization, and Personalized Learning Recommendations. These endpoints handled user requests, processed input data, generated AI-powered outputs, and returned structured JSON responses to the frontend interface. Error handling was also implemented to ensure proper validation when required input fields are missing. This backend API integration established smooth communication between the frontend interface and AI modules, enabling real-time educational assistance and efficient response generation within the EduGenie Learning Assistant.
