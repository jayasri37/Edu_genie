EduGenie – AI-Powered Educational Learning Assistant

OVERVIEW:
EduGenie is an AI-powered learning assistant developed during the SmartBridge Internship. It helps students by providing AI-based explanations, quizzes, summaries, and personalized learning support using Google Gemini AI.

TECHNOLOGIES:
- Python
- FastAPI
- HTML, CSS, JavaScript
- Google Gemini AI
- Git & GitHub

FEATURES:
- AI Question Answering
- Topic Explanation
- Quiz Generation
- Content Summarization
- Personalized Learning

RUN THE PROJECT:
```bash
ls
pip install -r requirements.txt
python -m uvicorn main:app --reload --port 8000
```

Open:
```
http://127.0.0.1:8000
```
